# processus
A one stop CLI for preprocessing any kind of data.
## Motivation
Often one finds himself overwhelmed by the huge array of data manipulation and processing commands offered by mulitple libraries. The motivation behind this project is to provide an one stop destination for all kind of data processing ranging from audio to video to text.
